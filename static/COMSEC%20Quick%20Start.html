COMSEC Quick Start

Updated
  • Use XMPP (and OTR encryption) for instant messaging.
    1. Download and install Jitsi.
    2. Use a company account or a personal account (sign up at one of the public servers).
  • Use PGP to secure e-mail.
    • Mac
      • Install GPG Suite.
      • Follow the simple official instructions or screencast.
      • Use Apple Mail with the included plugin, or go open source with Mozilla Thunderbird and the Enigmail add-on.
    • Windows
      • Install Gpg4win.
      • Consult one of many tutorials on YouTube.
      • Use Outlook with the included plugin—or go open source with Mozilla Thunderbird and the Enigmail add-on.
    • Linux
      • GPG comes pre-installed and you know how to work it.
    • Webmail (Gmail, etc.)
      1. Install GPG Suite (Mac) or Gpg4win (Windows).
      2. Use the Mailvelope extension for Firefox or Chrome.
      3. Get started.
    • Innovators & Early Adopters
      • Mailvelope is a free, open source, cross-platform, portable e-mail client that runs in your browser. User-friendly PGP encryption is under development.
